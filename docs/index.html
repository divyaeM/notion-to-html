The HTML export tool for Notion that actually works!


If you have ever used the default HTML export feature in Notion, you already know that the exported files are not very useful. They are not styled the same way as the original page and even miss some of your content.

With this tool you now can:

  • free yourself from the Notion lock-in
  • use Notion as your content management system
  • share your content with custom domain names

How to use

To be able to export a page with this tool, you must first set that page to be publicly accessible:

  1. Open the page you want to export
  2. Click on the Share button in the top right corner
  3. Toggle on the Share to web button
  4. Copy the link to the page by clicking on the Copy link button

Next you download the app and run it with the link you just copied.

If you are a software developer or familiar with the command line check out the CLI version of this app - else just click the download button below to continue.
