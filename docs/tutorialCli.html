Notion Snapshot: The HTML export tool for Notion that actually works!

... in the future, you'll be able to install the CLI via PIP, but for now, you'll have to clone the repo and install it manually.

1. Setup

If you havent already, you first need to install Python3 with Pip and Poetry.

Also if you are on Windows you additionally need to install WSL (Windows Subsystem for Linux) and Chrome.

This is how you install Chrome in WSL/Ubuntu: 


    sudo apt update && sudo apt upgrade -y
    wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
    sudo dpkg -i google-chrome-stable_current_amd64.deb
    sudo apt --fix-broken install
    rm -rf google-chrome-stable_current_amd64.deb

2. Cloning the repository


    git clone https://github.com/sueszli/notionSnapshot.git
    cd notionSnapshot

3. Installing the dependencies

To install the dependencies with Poetry, run: 


    poetry install --sync

But be careful because some dependencies might be missing and you might have to install them again with pip manually (e.g. pip install xyz).


4. Running the

Use the -h or --help flag when running the script to see all the options that are available. 


    python3 notionsnapshot https://eager-waterfall-308.notion.site/Loconotion-Example-03c403f4fdc94cc1b315b9469a8950ef

After runnning the script the exported files will be saved in the snapshots folder.
