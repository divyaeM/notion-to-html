The HTML export tool for Notion that actually works!


If you have ever used the default HTML export feature in Notion, you already know that the exported files are not very useful. They are not styled the same way as the original page and even miss some content.

Because the Notion API is just as helpful as the HTML export tool, we had to resort to web scraping.

With this tool you now can:

  • free yourself from the Notion lock-in
  • use Notion as your content management system
  • share your content with custom domain names

How to use

First you must set the page that you want to export to be publicly accessible:

  1. Open the page you want to export to HTML in Notion
  2. Click on the Share button in the top right corner
  3. Toggle the Share to web button (if not already enabled)
  4. Copy the link to the page by clicking on the Copy link button

Make sure to store the link you just copied somewhere safe, you will need it later.

Next you will need to install the app:

... we are planning to make this tool available through PIP, but for now you have to clone the repository and install the dependencies manually.

1. Setup

If you havent already, you first need to install:

  • Python 3
  • PIP
  • Poetry

Additionally if you are on a Windows machine you need to install:

  • WSL (Windows Subsystem for Linux)
  • All the dependencies above + Chrome inside your WSL

Installing Chrome in Ubuntu / WSL can be a bit tricky, but here is what worked for me: 


    sudo apt update && sudo apt upgrade -y
    wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
    sudo dpkg -i google-chrome-stable_current_amd64.deb
    sudo apt --fix-broken install
    rm -rf google-chrome-stable_current_amd64.deb

2. Cloning the repository

Next you have to clone the repository. 


    git clone https://github.com/sueszli/notionSnapshot.git
    cd notionSnapshot

3. Installing the dependencies

To install the dependencies with Poetry, run: 


    poetry install --sync

But be careful because some dependencies might still be missing after this step. You then might have to install them again with pip manually (e.g. pip install xyz).


4. Running the application

Use the -h or --help flag when running the script to see all the options that are available: 


    python3 notionsnapshot --help

You can for instance scrape the pages in dark mode by using the --dark-mode option or display the browser while scraping by using the --show-browser option.

Once you've made up your mind, you can run the script with the URL of the Notion page you want to scrape: 


    python3 notionsnapshot https://...notion.site/...

Alternatively you can also use one of our test pages:

  • All blocks: https://sueszli.notion.site/Notion-Snapshot-Test-Page-5ab361d19688436fb22f319e84b53a07
  • Templates: https://sueszli.notion.site/NotionSnapshot-Test-full-templates-cfbba0e4a3244e84b3242b137c2f640e

After runnning the script the exported files will be saved in the ./snapshots directory.
